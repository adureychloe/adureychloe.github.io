HackerRank 1 week preparation kit day2: Counting Sort

Posted on 2023-02-23，2 min read

题目描述

快速排序 的运行时间通常为 nlog(n) ，但是有没有一种算法可以更快地排序？一般来说，这是不可能的。大多数排序算法都是比较排序，即它们仅通过将元素相互比较来对列表进行排序。比较排序算法无法击败 nlog(n)（最坏情况）运行时间，因为 n*log(n) 表示知道每个元素的放置位置所需的最小比较次数。
另一种排序方法，就是计数排序（Counting Sort），不需要比较。相反，您创建一个整数数组，其索引范围涵盖要排序的数组中的整个值范围。每次原始数组中出现一个值时，您都会在该索引处增加计数器。最后，遍历您的计数数组，打印每个非零值索引的值该次数。

例子(Example)
arr=[1,1,3,2,1]
All of the values are in the range [0...3] , so create an array of zeros, result = [0,0,0,0] . The results of each iteration follow:
所有值都在范围内[0...3]，所以创建一个零数组，result = [0,0,0,0]. 每次迭代的结果如下：
i arr [ i ] result
0 1 [ 0 , 1 , 0 , 0 ]
1 1 [ 0 , 2 , 0 , 0 ]
2 3 [ 0 , 2 , 0 , 1 ]
3 2 [ 0 , 2 , 1 , 1 ]
4 1 [ 0 , 3 , 1 , 1 ]
The frequency array is [0,3,1,1] . These values can be used to create the sorted array as well: sorted = [1,1,1,2,3].
频率阵列是[0,3,1,1]. 这些值也可用于创建排序数组：sorted = [1,1,1,2,3].

对于本练习，始终返回一个包含 100 个元素的频率数组。

要点

数组始终是100个

题解

def countingSort(arr):
    arr.sort()
    result = [0]*100
    # Write your code here
    for i in arr:
        result[i]+=1        
    return result

if __name__ == '__main__':
    n = int(input().strip())
    arr = list(map(int, input().rstrip().split()))
    result = countingSort(arr)
    print(result)
