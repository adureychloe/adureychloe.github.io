牛客：有趣的数字

Posted on 2023-03-26，2 min read

题目描述

有n个数，两两组成二元组，输出相差最小有几对，相差最大有几对。
输入：个数，数据

例子：输入：
6
45 12 45 32 5 6
输出：
1 2

思路

先排序，用字典存储每个数字出现次数，若有重复的，则最小差就是0。最小差值不是0就是相邻差，如果没重复，就从后到前计算差值，排序，用count得到最小值次数。最大差值数量计算要注意用最大值个数乘以最小值个数。

题解


import sys

for line in sys.stdin:
    a = line.split()
    b = input().strip().split(' ')
    num = [int(i) for i in b]
    num.sort()
    Dict = {}
    for i in num:
        if i in Dict:
            Dict[i] += 1
        else:
            Dict[i] = 1

    res = 0
    for k in Dict.keys():
        if Dict[k] >= 2:
            temp2 = [i for i in range(Dict[k])]
            res += sum(temp2)  # 有相同的最小差就是0


    if res == 0: 
        # 没重复的情况，比如[1,2,3,9]这种
        temp3 = []
        for j in range(len(num)-1):
            temp3.append(num[j+1] - num[j])
        temp3.sort()
        # print()会换行，算例通不过，加了end就不会换行
        print(temp3.count(temp3[0]), end=" ")
    else:
        print(res, end=" ")
    num_max, num_min = Dict[num[-1]], Dict[num[0]]
    print(num_max*num_min)
