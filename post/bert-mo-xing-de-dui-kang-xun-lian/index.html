BERT模型的对抗训练

2022-07-08

一、对抗样本

让模型更加鲁棒的方法：

  • 对数据进行处理，将对攻击的防御放到数据处理过程中
  • 对模型输出的向量表征做一些转换，将防御放在模型的输出端
  • 将防御放在模型本身上

二、对抗训练经典算法

不同于图像输入是连续的，NLP输入为离散的单词序列，如何添加扰动？可以对embedding后的向量表征做扰动。


这种对抗训练方法叫做FGM（Fast Gradient Method），因为经过一步就得到对抗样本，容易导致扰动过大。
另一种方法PGD（Projected Gradient Descent）限定了扰动范围，对抗样本并不是一步就得到，而是沿着不同点的梯度走多步后得到：

三、代码实现

FGM:

class FGM():
    def __init__(self, model):
        self.model = model
        self.backup = {}

    def attack(self, epsilon=1., emb_name='embedding'):
        # 得到对抗样本
        # emb_name这个参数要换成你模型中embedding的参数名
        # 例如，self.embbeding = nn.Embedding(5000, 100)
        # 循环遍历模型所有参数
        for name, param in self.model.named_parameters():
            # 如果当前参数在计算中保留了对应的梯度信息，并且包含了模型中embedding的参数名
            if param.requires_grad and emb_name in name:
                # 把真实参数保存
                self.backup[name] = param.data.clone()
                # 对参数的梯度求范数
                norm = torch.norm(param.grad) # 默认为2范数
                # 如果范数不等于0并且norm中没有缺失值
                if norm != 0 and not torch.isnan(norm):
                    # 计算扰动，param.grad / norm = 单位向量，起到了和sgn(param.grad)一样的作用
                    r_at = epsilon * param.grad / norm
                    # 在原参数上添加扰动
                    param.data.add_(r_at)

    def restore(self, emb_name='embbedding'):
        # 将模型原本的参数复原
        # emb_name这个参数要换成你模型中embedding的参数名
        for name, param in self.model.named_parameters():
            if param.requires_grad and emb_name in name: 
                # 断言
                assert name in self.backup
                # 取出模型真实参数
                param.data = self.backup[name]
        # 清空backup
        self.backup = {}

# 训练函数
def train():
    model.train()
    fgm = FGM(model)
    total_train_loss = 0
    iter_num = 0
    total_iter = len(train_loader)
    for batch in train_loader:
        # 正向传播
        optim.zero_grad()
        
        input_ids = batch['input_ids'].to(device)
        attention_mask = batch['attention_mask'].to(device)
        labels = batch['labels'].to(device)
        
        outputs = model(input_ids, attention_mask=attention_mask, labels=labels)
        loss = outputs[0]
        total_train_loss += loss.item()
        
        # 反向传播，得到正确的梯度信息
        loss.backward()
        torch.nn.utils.clip_grad_norm_(model.parameters(), 1.0)
        
        # 产生对抗样本
        # 再次正向传播
         
        # 对抗训练
        fgm.attack() # embedding被修改了
        loss_adv = model(input_ids, attention_mask=attention_mask, labels=labels)
        loss_adv[0].backward() # 反向传播，在正常的grad基础上，累加对抗训练的梯度
        fgm.restore() # 恢复Embedding的参数
        
        # 参数更新
        optim.step()
        scheduler.step()

        iter_num += 1
        if(iter_num % 10==0):
            print("epoth: %d, iter_num: %d, loss: %.4f, %.2f%%" % (epoch, iter_num, loss.item(), iter_num/total_iter*100))
        
    print("Epoch: %d, Average training loss: %.4f"%(epoch, total_train_loss/len(train_loader)))
