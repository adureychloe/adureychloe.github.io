Leetcode 92: Reverse Linked List II

Posted on 2023-08-03，2 min read

题目描述

给定一个单链表和两个大小数left和right，反转链表从left到right的位置，返回反转链表。

思路

反转前N个节点

思路和反转整个链表差不多：

ListNode* successor = nullptr;

// 反转以 head 为起点的 n 个节点，返回新的头结点
ListNode* reverseN(ListNode* head, int n) {
    if (n == 1) {
        successor = head->next;
        return head;
    }
    ListNode* last = reverseN(head->next, n - 1);
    head->next->next = head;
    head->next = successor;
    return last;
}

区别：

  • 送入N，base case变为N==1。反转一个元素就是它本身，同时记录后驱节点。
  • 反转后head节点后面变为后驱节点。

反转链表的一部分

ListNode* reverseBetween(ListNode* head, int m, int n)

首先，如果 m == 1, 相当于反转开头n个元素。

如果 m!=1，就要用递归让其慢慢变成m=1，即变成反转链表开头N个元素。

ListNode* reverseBetween(ListNode* head, int m, int n) {
    // base case
    if (m == 1) {
        return reverseN(head, n);
    }
    // 前进到反转的起点触发 base case
    head->next = reverseBetween(head->next, m - 1, n - 1);
    return head;
}
