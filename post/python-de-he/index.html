python的*和**

Posted on 2023-08-02，2 min read

在python中，* 和 ** 是用于处理可变数量参数的特殊符号。

*

  • 在函数定义时，*args 用于表述函数可以接受任意数量的位置参数，这些参数会被打包成一个元组供函数内部使用。
def foo(*args):
    for arg in args:
        print(arg)
foo(1,2,3) # output: 1 2 3
  • 在函数调用时，用于解包列表或元组中元素，将它们作为位置参数传递给函数。
def bar(x,y,z):
    print(x,y,z)

values = [1,2,3]
bar(*values)

**

  • 在函数定义时，**kwargs用于表述函数可以接受任意数量的关键字参数。这些参数会被打包成一个字典供函数内部使用。
def foo(**kwargs):
    for key, value in kwargs.items():
        print(key, value)
foo(a=1, b=2, c=3)
  • 在函数调用时，**可以用于解包字典，将键值对作为关键字参数传递给函数。
def bar(x,y,z):
    print(x,y,z)
values={'x':1,'y':2, 'z':3}
bar(**values) # output: 1 2 3

注意事项

  • 函数定义时，通常放在参数最后
  • 函数调用时，通常放在参数之前
  • 当同时使用*args和**kwargs时，前者必须放在后者之前。
def example(a,b,*args,x=0,y=0,**kwargs):
    print(a,b)
    print(args) # 元组
    print(x,y)
    print(kwargs) # 字典

example(1,2,3,4,5,x=6,y=7,p=8,q=9)
# output:
# 1 2
# (3,4,5)
# 6 7
# {'p':8, 'q':9}
