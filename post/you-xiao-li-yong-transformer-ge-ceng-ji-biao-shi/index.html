原文：Utilizing Transformer Representations Efficiently
该笔记详细记录了利用Transformer各层上下文表示的方法，所有方法都有实例代码，能直接应用到实验中。

Introduction

微调的典型方式：在Transformer之后只附加一个额外的输出层，用于下游任务或模型的后端部分，它将预训练的语言模型的最后一层的表示作为默认输入。
然而由于Transformer的多层结构，不同层会捕获不同层次的表征，学习到丰富的语义信息，即在底层有表面特征，中层有句法特征，高层有语义特征。

BERT的作者通过将不同的向量组合作为输入特征输入到一个用于命名实体识别任务的BiLSTM中，并观察所得到的F1分数，来测试单词嵌入策略。发现最后四层的串联产生了最好的结果。这就部分证明了bert的不同层编码了不同的信息。

Overview

在HuggingFace的Transformers中输入input_ids和attention_mask后有2个主要输出（配置后是3个）：

  • pooler output (batch size, hidden size) - 序列中第一个token的最后一层隐藏状态
  • last hidden state (batch size, seq Len, hidden size) 在最后一层输出的隐藏状态序列
  • hidden states (n layers, batch size, seq Len, hidden size) - 所有层和所有id的隐藏状态

Preliminary Setup

import包：

import gc; gc.enable()
from IPython.display import clear_output
import pandas as pd
train = pd.read_csv('../input/commonlitreadabilityprize/train.csv')

Pooler Output

Introduction

这通常是transformer的第二层输出和最简单的一个输出。Pooler Output是序列中第一个token（分类标记）的最后一层隐藏状态，由线性层和Tanh激活函数进一步处理。线性层的权重是在预训练期间从NSP（分类）目标中训练出来的。
Last hidden state被传递到pooler中，这将返回pooler output。我们可以通过在模型配置中设置添加池化层为False并将其传递给模型来停用池化输出。

Implementation

利用Pooler output到下游任务。

import torch
import torch.nn as nn
import transformers
from transformers import (
    AutoModel, AutoConfig, 
    AutoTokenizer, logging
)
logging.set_verbosity_error()
logging.set_verbosity_warning()

train_text = train['excerpt'][:16].tolist()
max_seq_length = 256
_pretrained_model = 'roberta-base'

config = AutoConfig.from_pretrained(_pretrained_model)
model = AutoModel.from_pretrained(_pretrained_model, config=config)
tokenizer = AutoTokenizer.from_pretrained(_pretrained_model)

clear_output()

features = tokenizer.batch_encode_plus(
    train_text,
    add_special_tokens=True,
    padding='max_length',
    max_length=max_seq_length,
    truncation=True,
    return_tensors='pt',
    return_attention_mask=True
)

with torch.no_grad():
    outputs = model(features['input_ids'], features['attention_mask'])
pooler_output = outputs[1]
logits = nn.Linear(config.hidden_size, 1)(pooler_output) # regression head

print(f'Pooler Output Shape: {pooler_output.detach().numpy().shape}')
print(f'Logits Shape: {logits.detach().numpy().shape}')

del config, model, tokenizer, features, outputs
gc.collect();

Pooler Output Shape: (16, 768)
Logits Shape: (16, 1)

Last Hidden State Output

Introduction

这是模型的第一输出也是默认输出。Last Hidden State Output是模型最后一层输出的隐藏状态的序列。输出通常是[batch, maxlen, hidden_state]，对于[CLS]token，它可以缩小为[batch, 1, hidden_state]，因为[CLS]token是序列中的第一个token。这里，[batch, 1, hidden_state]可以等同于[batch, hidden_state]。

CLS Embeddings

Introduction

由于Transformers是上下文模型，其思想是[CLS]token将捕获整个上下文，并足以完成简单的下游任务，如分类。因此，对于使用句子表示进行分类这样的任务，可以使用[batch，hidden_state]。

Implementation

利用cls embeddings输出到下游任务。

import torch
import torch.nn as nn
import transformers
from transformers import (
    AutoModel, AutoConfig, 
    AutoTokenizer, logging
)
logging.set_verbosity_error()
logging.set_verbosity_warning()

train_text = train['excerpt'][:16].tolist()
max_seq_length = 256
_pretrained_model = 'roberta-base'

config = AutoConfig.from_pretrained(_pretrained_model)
model = AutoModel.from_pretrained(_pretrained_model, config=config)
tokenizer = AutoTokenizer.from_pretrained(_pretrained_model)

clear_output()

features = tokenizer.batch_encode_plus(
    train_text,
    add_special_tokens=True,
    padding='max_length',
    max_length=max_seq_length,
    truncation=True,
    return_tensors='pt',
    return_attention_mask=True
)

with torch.no_grad():
    outputs = model(features['input_ids'], features['attention_mask'])
last_hidden_state = outputs[0]
cls_embeddings = last_hidden_state[:, 0]

print(f'Last Hidden State Output Shape: {last_hidden_state.detach().numpy().shape}')

Last Hidden State Output Shape: (16, 256, 768)
CLS Embeddings Output Shape: (16, 768)
Logits Shape: (16, 1)

Mean Pooling

Introduction

我们也可以考虑 last hidden state [batch, maxlen, hidden_state]，即跨maxlen维度的平均值，以获得平均/均值嵌入。
有多种不同的方法来做到这一点。我们可以简单地采用 torch.mean(last_hidden_state, 1)，但我们要实现不同的东西。我们将利用attention masks，这样我们就可以忽略padding tokens，这是实现平均嵌入的一个更好的方法。

Implementation

利用平均池化输出到下游任务。

import torch
import torch.nn as nn
import transformers
from transformers import (
    AutoModel, AutoConfig, 
    AutoTokenizer, logging
)
logging.set_verbosity_error()
logging.set_verbosity_warning()

train_text = train['excerpt'][:16].tolist()
max_seq_length = 256
_pretrained_model = 'roberta-base'

config = AutoConfig.from_pretrained(_pretrained_model)
model = AutoModel.from_pretrained(_pretrained_model, config=config)
tokenizer = AutoTokenizer.from_pretrained(_pretrained_model)

clear_output()

features = tokenizer.batch_encode_plus(
    train_text,
    add_special_tokens=True,
    padding='max_length',
    max_length=max_seq_length,
    truncation=True,
    return_tensors='pt',
    return_attention_mask=True
)

with torch.no_grad():
    outputs = model(features['input_ids'], features['attention_mask'])
last_hidden_state = outputs[0]
attention_mask = features['attention_mask']
  • step1：将注意力掩码从[batch_size, max_len]扩展到[batch_size, max_len, hidden_size]。
  • step2：沿max_len轴对embeddings求和，所以现在我们有[batch_size, hidden_size]。
  • step3：沿max_len轴对Mask求和。这样做是为了让我们能够忽略padding tokens。
  • step4：取平均值。
input_mask_expanded = attention_mask.unsqueeze(-1).expand(last_hidden_state.size()).float()
sum_embeddings = torch.sum(last_hidden_state * input_mask_expanded, 1)
sum_mask = input_mask_expanded.sum(1)
sum_mask = torch.clamp(sum_mask, min=1e-9)
mean_embeddings = sum_embeddings / sum_mask
logits = nn.Linear(config.hidden_size, 1)(mean_embeddings) # regression head

print(f'Last Hidden State Output Shape: {last_hidden_state.detach().numpy().shape}')
print(f'Mean Embeddings Output Shape: {mean_embeddings.detach().numpy().shape}')
print(f'Logits Shape: {logits.detach().numpy().shape}')

del config, model, tokenizer, features
gc.collect();

Last Hidden State Output Shape: (16, 256, 768)
Mean Embeddings Output Shape: (16, 768)
Logits Shape: (16, 1)

Max Pooling

Introduction

与平均池化类似，我们将考虑 last hidden state [batch, maxlen, hidden_state]，然后在maxlen维度上取最大值，得到最大池嵌入。
也有多种不同的方法来做这个。我们可以简单地采取Torch.max(last_hidden_state, 1)，但我们将通过利用attention masks来实现类似于平均池化的方法。

Implementation

利用最大池化embeddings输出到下游任务。

import torch
import torch.nn as nn
import transformers
from transformers import (
    AutoModel, AutoConfig, 
    AutoTokenizer, logging
)
logging.set_verbosity_error()
logging.set_verbosity_warning()

train_text = train['excerpt'][:16].tolist()
max_seq_length = 256
_pretrained_model = 'roberta-base'

config = AutoConfig.from_pretrained(_pretrained_model)
model = AutoModel.from_pretrained(_pretrained_model, config=config)
tokenizer = AutoTokenizer.from_pretrained(_pretrained_model)

clear_output()

features = tokenizer.batch_encode_plus(
    train_text,
    add_special_tokens=True,
    padding='max_length',
    max_length=max_seq_length,
    truncation=True,
    return_tensors='pt',
    return_attention_mask=True
)

with torch.no_grad():
    outputs = model(features['input_ids'], features['attention_mask'])
last_hidden_state = outputs[0]
attention_mask = features['attention_mask']

input_mask_expanded = attention_mask.unsqueeze(-1).expand(last_hidden_state.size()).float()
last_hidden_state[input_mask_expanded == 0] = -1e9  # Set padding tokens to large negative value
max_embeddings = torch.max(last_hidden_state, 1)[0]
logits = nn.Linear(config.hidden_size, 1)(max_embeddings) # regression head

print(f'Last Hidden State Output Shape: {last_hidden_state.detach().numpy().shape}')
print(f'Max Embeddings Output Shape: {max_embeddings.detach().numpy().shape}')
print(f'Logits Shape: {logits.detach().numpy().shape}')

del config, model, tokenizer, features
gc.collect();

Last Hidden State Output Shape: (16, 256, 768)
Max Embeddings Output Shape: (16, 768)
Logits Shape: (16, 1)

Mean-Max Pooling

Introduction

Mean Max Pooling或Mean Max Head是许多kernels和竞赛中最常用的技术。我们首先找到均值和最大池化嵌入，然后将其串联起来，得到一个两倍于hidden size的最终表示。上图b显示了这是如何做到的。
这里我们将采用Kaggle的方法，但也可以像前面的方法那样找到均值和最大值嵌入，然后连接起来，得到一个最终的结果向量，其形状也是一样的，但不会考虑padding tokens。

Implementation

利用平均-最大池化embeddings输出到下游任务。

import torch
import torch.nn as nn
import transformers
from transformers import (
    AutoModel, AutoConfig, 
    AutoTokenizer, logging
)
logging.set_verbosity_error()
logging.set_verbosity_warning()

train_text = train['excerpt'][:16].tolist()
max_seq_length = 256
_pretrained_model = 'roberta-base'

config = AutoConfig.from_pretrained(_pretrained_model)
model = AutoModel.from_pretrained(_pretrained_model, config=config)
tokenizer = AutoTokenizer.from_pretrained(_pretrained_model)

clear_output()

features = tokenizer.batch_encode_plus(
    train_text,
    add_special_tokens=True,
    padding='max_length',
    max_length=max_seq_length,
    truncation=True,
    return_tensors='pt',
    return_attention_mask=True
)

with torch.no_grad():
    outputs = model(features['input_ids'], features['attention_mask'])
last_hidden_state = outputs[0]

mean_pooling_embeddings = torch.mean(last_hidden_state, 1)
_, max_pooling_embeddings = torch.max(last_hidden_state, 1)
mean_max_embeddings = torch.cat((mean_pooling_embeddings, max_pooling_embeddings), 1)
logits = nn.Linear(config.hidden_size*2, 1)(mean_max_embeddings) # twice the hidden size

print(f'Last Hidden State Output Shape: {last_hidden_state.detach().numpy().shape}')
print(f'Mean-Max Embeddings Output Shape: {mean_max_embeddings.detach().numpy().shape}')
print(f'Logits Shape: {logits.detach().numpy().shape}')

del config, model, tokenizer, features
gc.collect();

Last Hidden State Output Shape: (16, 256, 768)
Mean-Max Embeddings Output Shape: (16, 1536)
Logits Shape: (16, 1)

Conv-1D Pooling

Introduction

使用conv1d层来过滤不想要的特征。这里内核大小为2意味着它考虑了两个token来进行跨特征的卷积。conv1d的主要思想是进行跨token的滑动，通过探索所有768个特征找到最重要的token。

Implementation

利用conv1d embedding输出到下游任务。

import torch
import torch.nn as nn
import torch.nn.functional as F
import transformers
from transformers import (
    AutoModel, AutoConfig, 
    AutoTokenizer, logging
)
logging.set_verbosity_error()
logging.set_verbosity_warning()

train_text = train['excerpt'][:16].tolist()
max_seq_length = 256
_pretrained_model = 'roberta-base'

config = AutoConfig.from_pretrained(_pretrained_model)
model = AutoModel.from_pretrained(_pretrained_model, config=config)
tokenizer = AutoTokenizer.from_pretrained(_pretrained_model)

clear_output()

features = tokenizer.batch_encode_plus(
    train_text,
    add_special_tokens=True,
    padding='max_length',
    max_length=max_seq_length,
    truncation=True,
    return_tensors='pt',
    return_attention_mask=True
)

with torch.no_grad():
    outputs = model(features['input_ids'], features['attention_mask'])
last_hidden_state = outputs[0]


# first define layers
cnn1 = nn.Conv1d(768, 256, kernel_size=2, padding=1)
cnn2 = nn.Conv1d(256, 1, kernel_size=2, padding=1)

last_hidden_state = last_hidden_state.permute(0, 2, 1)
cnn_embeddings = F.relu(cnn1(last_hidden_state))
cnn_embeddings = cnn2(cnn_embeddings)
logits, _ = torch.max(cnn_embeddings, 2)

print(f'Last Hidden State Output Shape: {last_hidden_state.detach().numpy().shape}')
print(f'CNN Embeddings Output Shape: {cnn_embeddings.detach().numpy().shape}')
print(f'Logits Shape: {logits.detach().numpy().shape}')

del config, model, tokenizer, features
gc.collect();

Last Hidden State Output Shape: (16, 768, 256)
CNN Embeddings Output Shape: (16, 1, 258)
Logits Shape: (16, 1)
