python直接赋值、浅拷贝和深拷贝

Posted on 2023-08-16，4 min read

python 在赋值时如果不是不可变对象（数字、字符、元组），默认是直接赋值，就是对对象的引用。改变新值旧值也会发生改变。

要让新值不影响旧值，就要使用copy浅拷贝和deepcopy深拷贝。

  • 浅拷贝(copy)：拷贝父对象，不会拷贝对象的内部的子对象。

  • 深拷贝(deepcopy)： copy 模块的 deepcopy 方法，完全拷贝了父对象及其子对象。

>>>a = {1: [1,2,3]}
>>> b = a.copy()
>>> a, b
({1: [1, 2, 3]}, {1: [1, 2, 3]})
>>> a[1].append(4)
>>> a, b
({1: [1, 2, 3, 4]}, {1: [1, 2, 3, 4]})

>>>import copy
>>> c = copy.deepcopy(a)
>>> a, c
({1: [1, 2, 3, 4]}, {1: [1, 2, 3, 4]})
>>> a[1].append(5)
>>> a, c
({1: [1, 2, 3, 4, 5]}, {1: [1, 2, 3, 4]})

解析
1、b = a: 赋值引用，a 和 b 都指向同一个对象。

2、b = a.copy(): 浅拷贝, a 和 b 是一个独立的对象，但他们的子对象还是指向统一对象（是引用）。

3、b = copy.deepcopy(a): 深度拷贝, a 和 b 完全拷贝了父对象及其子对象，两者是完全独立的。

import copy
a = [1, 2, 3, 4, ['a', 'b']] #原始对象
 
b = a                       #赋值，传对象的引用
c = copy.copy(a)            #对象拷贝，浅拷贝
d = copy.deepcopy(a)        #对象拷贝，深拷贝
 
a.append(5)                 #修改对象a
a[4].append('c')            #修改对象a中的['a', 'b']数组对象
 
print( 'a = ', a )
print( 'b = ', b )
print( 'c = ', c )
print( 'd = ', d )

输出结果为：

('a = ', [1, 2, 3, 4, ['a', 'b', 'c'], 5])
('b = ', [1, 2, 3, 4, ['a', 'b', 'c'], 5])
('c = ', [1, 2, 3, 4, ['a', 'b', 'c']])
('d = ', [1, 2, 3, 4, ['a', 'b']])

但如果是不可变对象，包括数字、字符、元组、类的实例化对象等，这些对象在创建后不能被修改。当你对其做赋值操作时，两者都指向同一对象，一方的改变不会影响另一个变量，改变的那个变量会指向新的对象。但是对类来说，如果你改了元素的属性，那两个都会变（因为都指向同一位置）。
如：

x=10
y=x # 两者都指向10
x=20 # x指向新对象
print(x,y) # 20,10

可以用两个例子来更深刻的理解：

class ListNode():
    def __init__(self, x):
        self.val = x
        self.next = None
a = ListNode(1)
b = ListNode(2)
a.next = b

def func(head:ListNode)->ListNode:
    cur = head # 指向同一个位置
    for i in range(1):
        cur.next = ListNode(3) # 此位置的属性改变，两个都改变
    print(head.next) # ListNode(3)

func(a)

class ListNode():
    def __init__(self, x):
        self.val = x
        self.next = None
a = ListNode(1)
b = ListNode(2)
a.next = b

def func(head:ListNode)->ListNode:
    cur = head # 指向同一个位置
    for i in range(1):
        cur = ListNode(3) # cur指向新的位置，不影响head
    print(head.next) # ListNode(2)

func(a)
