HackerRank ProjectEuler #1 : Multiples of 3 and 5

Posted on 2023-02-22，2 min read

题目描述

10以下的自然数中，属于3和5的倍数的有3,5,6和9，它们之和是23。找出N以下的自然数中，属于3和5的倍数的数字之和。

题解1：暴力求解

t = int(input().strip())
sum = 0
for a0 in range(t): # index 
    n = int(input().strip()) # data
    for i in range(n):
        if i % 3 == 0 or i % 5 == 0:
            print(i)
            sum += i       
    print(sum)

没过测试，时间复杂度太高。

题解2：等差级数

不遍历，而是求出3的倍数和5的倍数相加（因为有重叠所以还要减去15的倍数）。
如果N=1000，3的倍数之和为3+6+9+12+…+999=3*(1+2+3+4+…+333)
5的倍数之和为5+10+15+…+995=5*(1+2+…+199)，
可以发现就是转换为3或5乘上一串等差数列（d=1），数列的第一个数是1，最后一个数是（N-1)//3或5，//代表整除向下取整，N-1是因为不包含N。数列的和为： Sn=12n(a1+an)S_{n} = \frac{1}{2} n(a_{1}+a_{n})Sn​=21​n(a1​+an​)。

def sum(a, n):
    N = (n-1)// a
    return a*(N*(N+1)) // 2

t = int(input().strip())
result = []
# memo = []
for a0 in range(t):
    n = int(input().strip()) # data
    result = sum(3,n) + sum(5,n) - sum(15,n)
    print(result)
