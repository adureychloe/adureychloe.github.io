HackerRank Dynamic Programming: The Coin Change Problem

Posted on 2023-03-29，2 min read

题目描述

给定一个数amount和一些固定面值的硬币c[i]，硬币数量无限，输出有多少种组合方式可以得到那个数。

思路

这是个动态规划问题，因为它具有最优子结构，也就是子问题之间相互独立。子问题就是当amount=10时，如果你知道 amount = 10 - c[i] 时的组合方式 n，那10的组合方式就是 n（再加上c[i]就是10）。
设dp[i]：输入amount为i，有dp[i]种组合方式。
base case：dp[0]=1（输入0，有1种组合方式（只有0））
状态：组合方式个数
选择：不同硬币
对于不同硬币，分别求出各自的组合方式个数，然后相加。

题解


import math
import os
import random
import re
import sys

#
# Complete the 'getWays' function below.
#
# The function is expected to return a LONG_INTEGER.
# The function accepts following parameters:
#  1. INTEGER n
#  2. LONG_INTEGER_ARRAY c
#

def getWays(n, c):
    # Write your code here
    dp = [0] * (n+1)
    # base case
    dp[0] = 1
    
    # main logic
    for i in range(len(c)):
        for j in range(c[i], n+1):
            dp[j] += dp[j-c[i]] # 减去当前硬币的组合方式个数
        print(c[i], dp)
        
    return dp[n]
