排序算法

Posted on 2023-08-30，2 min read

分类

可分为内部排序和外部排序，分别在内存中排序和需要访问外存。

时间复杂度

  • O(n2n^2n2)：各类简单排序：直接插入、直接选择、冒泡排序
  • O(nlog2n): 快排、堆排序、归并排序
  • O(n(1+a))：希尔排序
  • O(n)：基数排序、桶排序

稳定性

稳定性指排序后两个相等键值的顺序和排序之前它们的顺序相同。

  • 稳定：冒泡排序、插入排序、归并排序和基数排序
  • 不稳定：选择排序、快速排序、希尔排序、堆排序

快速排序

先从数组中选取一个基准点pivot，将待排序集合中所有小于pivot的元素移至左侧，大于的到右侧，然后将此时的指针指向元素与pivot交换，定义pivot为已排序部分，分别对左边和右边继续排序。

如果pivot选取第一或最后一个元素，平均时间复杂度是O(nlogn)，当遇到数组有序的情况下出现最坏时间复杂度O(n^2)，因为此时pivot以外的元素都在一边，浪费了另一个区间的作用。

因为递归（栈），快排空间复杂度是O(logn)。

Q：为什么pivot放入左侧时，必须先平移右边指针？

A：因为如果先平移左边，跳出两个条件的主动权掌握在左指针上，也就是最后停留的位置是一个大于pivot的数（此时左指针跳出循环，右指针无法再继续往左，也跳出），此时交换pivot和该数，就错了。

def sortArray(self, nums):
    def quicksort(l,r):
        if l>=r:
            return
        pivot, i, j = nums[l], l, r
        while i<j:
            while i<j and nums[j]>=pivot:
                j-=1
            while i<j and nums[i]<=pivot:
                i+=1
            nums[i], nums[j] = nums[j], nums[i]
        nums[l], nums[i] = nums[i], nums[l]
        quicksort(l,i-1)
        quicksort(i+1,r)
    quicksort(0, len(nums)-1)
    return nums
