pandas合并两个df

Posted on 2023-02-08，1 min read

合并两个df：

（1）train和test两个表，相关列为"cfips"。要将train的三列按照cfips连到test。先将包括cfips的三列变成字典，index为cfips，采用map函数映射到test上。

state_dict = train[['cfips', 'state', 'county']]
state_dict = state_dict.set_index('cfips')
state_dict = state_dict.drop_duplicates()
state_dict = state_dict.to_dict()

test['state'] = test['cfips'].map(state_dict['state'])
test['county'] = test['cfips'].map(state_dict['county'])

（2）df_all和df_census两个表，要将df_census按照cfips连到df_all后，df_census的index是cfips。首先将df_all的index也变成cfips，然后用df_all[df_census.columns] = df_census。（df_census和df_all维度不一样，df_all里一个cfips有好几列，df_census里一个cfips只有一列）

df_all = df_all.reset_index()
df_all = df_all.set_index('cfips') # in order to add df_census

df_all[df_census.columns] = df_census
