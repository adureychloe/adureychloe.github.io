一、同花顺金融领域事件关系抽取

主要对金融文本中因果事件的抽取，面向的是句子级别，抽取形式为(cuase, influence, effect)三元组，其中关系只有influence一种类型，cause与effect都为金融领域的事件；

例如：中国作为猪肉消费大国，猪肉价格上涨引起政府的高度重视。

抽取：[(猪肉价格上涨, influence, 政府的高度重视)]

评价指标

(1)因果事件抽取F1值；(2)因果关系三元组抽取F1值；(3)因果关系三元组easy版F1值；

三个指标的平均值作为最终排名指标

因为考虑有些预测出三元组就相差几个token，按(2)方式计算过于严格，所以提出easy版F1值，将这类情况也视为预测正确；具体判断依据为：当预测出的三元组中的因果事件与标准中因果事件的最长公共子序列系数都超过0.9，也当作预测正确；

F值按Micro思路进行计算：

        precision = correct_num / predict_num
        recall = correct_num / gold_num
        F1 = (2 * precision * recall) / (precision + recall）

其中correct_num, predict_num, gold_num分别代表预测正确的三元组数量、预测出的三元组数量，标准中三元组数量。

关于最长公共子序列是调用pylcs包进行计算，具体如下：

def fsim(s1, s2):
    lcs = pylcs.lcs(s1, s2)
    return 2. * lcs / (len(s1) + len(s2))
