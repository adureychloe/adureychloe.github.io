python变量赋值问题：可变或不可变对象

如果直接赋值，就是对另一个变量的内存地址的拷贝，也就是说和原来的变量指向同一个内存地址。

  • （1）当赋值不可变对象时，对其中一个变量改变不会影响另一个，改变的变量相当于赋了一个新的内存地址。如元组。
  • （2）当赋值可变对象时，会一起改变，内存地址不变。如列表。如果要生成完全新的对象，应该使用deepcopy：
import copy
data1=copy.deepcopy(data)

python 的 / 和 //

都是除法，// 是整数除法，向下取整， / 是浮点数除法，返回浮点结果。

python字符串变列表

如果想把字符串每个字符变成列表，直接用list。

num = 10
strNum = list(str(num)) # ['1','0']
