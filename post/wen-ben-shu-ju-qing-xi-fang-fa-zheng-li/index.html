删除空的成对符号

def remove_empty_paired_punc(in_str):
    return in_str.replace('（）',' ').replace('《》',' ').replace('【】',' ').replace('[]', ' ')

删除html标签

def remove_html_tags(in_str):
    html_pattern = re.compile(r'<[^>]+>', re.S)
    return html_pattern.sub(' ', in_str)

删除不可见控制字符

def remove_control_chars(in_str):
    control_chars = ' '.join(map(chr, list(range(0, 32)) + list(range(127, 160))))
    control_chars = re.compile(' [%s] ' % re.escape(control_chars))
    return control_chars.sub(' ', in_str)

英文替换非标准字符

text = re.sub(r"[^a-zA-Z0-9]", " ", text)

去除停用词

stoplist = stopwords.words('english')
text = [word for word in text if word not in stoplist]

还原词干

text = [PorterStemmer().stem(word) for word in text]

添加开始结束符

text.append(" eos") # 添加结束符
text = ["bos"] + text

中文替换非标准字符

text = re.sub(r"[a-zA-Z0-9-，。“”（）]", " ", text)

字符替换

str.maketrans()

若输入2个参数，二者必须为长度相等且一一对应的字符串。第一个是需要转换的字符，第二个是转换目标。

from string import maketrans   # 必须调用 maketrans 函数。

intab = "aeiou"
outtab = "12345"
trantab = maketrans(intab, outtab)

str = "this is string example....wow!!!";
print str.translate(trantab);

th3s 3s str3ng 2x1mpl2....w4w!!!

若输入3个参数，则它必须是一个字符串，其中的字符将在输出字典中被映射到 None，也就是所有该字符都被替换为None。

# 去除标点符号
punct = str.maketrans("","", string.punctuation)

pandas文本Counter化

对list类型的文本集合不能直接Counter：

test = train_fn['full_text'][0:3]
cnt = Counter(test)

应该分开来添加：

def counter (text):
    cnt = Counter()
    for msgs in text:
        for msg in msgs:
            cnt [msg] += 1
    return cnt
text_cnt = counter(train["full_text_clean"])

这里msgs是token list，msg是token。
