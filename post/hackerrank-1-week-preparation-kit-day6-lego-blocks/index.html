HackerRank 1 week preparation kit day6: Lego Blocks

Posted on 2023-03-21，4 min read

题目描述

有四种砖块（depth x height x width）：1x1x1, 1x1x2, 1x1x3, 1x1x4。
使用这些砖块砌墙n×mn \times mn×m，n是height，m是width。
有三个限制：

  • 墙不能有洞
  • 墙的结构坚实，在所有行之间不会有竖直的一条线
  • 砖块必须水平放置

输入n和m，求有几种放法。

思路

  • 先考虑简单的情况：没有第二个限制，只看每层的可能数，然后乘以行数。这一步用动态规划，base case就是砖块分别有1，2，3，4时的情况，大于4的砖块都可以分为这些base case之和（K(N)=K(N−1)+K(N−2)+K(N−3)+K(N−4)K(N) = K(N-1) + K(N-2) + K(N-3) + K(N-4)K(N)=K(N−1)+K(N−2)+K(N−3)+K(N−4)）。用一个列表存储可能放法的数量，index和砖块数一一对应（0不算）
  • 然后就是加上第二个限制，即将全部情况数减去违反情况数。设违反的列为x（x的边是横贯砖块的一条直线），x把全部列分成了两部分，一边放不违反的，一边随便放，避免重复计算。

左边始终是不违反的（正常 - 违反），右边随意（如果有违反的随着x的递增后面也会遇到，违反的迟早会变成左边被减去）。然后左边的可能数乘以右边的可能数就是此时有违反线的情况下的可能数，对i列的所有违反情况进行相加，添加到i对应的违反情况列表中。
最后将m列的所有情况减去m列的违反情况，就是最终结果。

题解

import math
import os
import random
import re
import sys

#
# Complete the 'legoBlocks' function below.
#
# The function is expected to return an INTEGER.
# The function accepts following parameters:
#  1. INTEGER n
#  2. INTEGER m
#

def legoBlocks(n, m):
    # Write your code here
    
    MOD = (10**9 +7)

    # Step 1: O(W)       
    # The number of combinations to build a single row
    # As only four kinds of sizes, so
    # base case: 
    # if width is 0, combination is 1
    # if width is 1, combination is 1
    # if width is 2, combination is 2
    # if width is 3, combination is 4
    row_combinations = [1, 1, 2, 4, 8]
    
    # Build row combinations up to current wall's width
    while len(row_combinations) <= m: 
        row_combinations.append(sum(row_combinations[-4:]) % MOD)
    
    # Step 2: O(W)
    # Compute total combinations 
    # for constructing a wall of height N of varying widths
    total = [pow(c, n, MOD) for c in row_combinations]
    
    # Step 3: O(W^2)
    # Find the number of unstable wall configurations 
    # for a wall of height N of varying widths
    unstable = [0, 0]
    
    # Divide the wall into left part and right part,
    # and calculate the combination of left part and right part.
    # From width is 2, we start to consider about violation.
    for i in range(2, m + 1):
        # i: current total width
        # j: left width
        # i - j: right width
        # f: (left part - previous vertical violation)*right part
        f = lambda j: (total[j] - unstable[j]) * total[i - j]
        result = sum(map(f, range(1, i)))
        unstable.append(result % MOD)
    
    # Print the number of stable wall combinations
    return (total[m] - unstable[m]) % MOD

if __name__ == '__main__':
    fptr = open(os.environ['OUTPUT_PATH'], 'w')

    t = int(input().strip())

    for t_itr in range(t):
        first_multiple_input = input().rstrip().split()

        n = int(first_multiple_input[0])

        m = int(first_multiple_input[1])

        result = legoBlocks(n, m)

        fptr.write(str(result) + '\n')

    fptr.close()
