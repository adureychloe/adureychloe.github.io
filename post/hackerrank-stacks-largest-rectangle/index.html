HackerRank Stacks: Largest Rectangle

Posted on 2023-04-01，2 min read

题目描述

有一些二维建筑,高度为h[i] , 如果把相邻的 k 个建筑连接起来,就能形成稳固的长方形区域 k×min(h[i],h[i+1],...h[i+k−1])k \times min(h[i],h[i+1],...h[i+k-1])k×min(h[i],h[i+1],...h[i+k−1]) (把上层多出来部分切掉), 找到最大的区域.

输入建筑数量n和高度h.

例子:

思路

首先新建栈来存储递增的楼, 用i来指向下一层楼. 直到碰到矮的楼.此时分别计算递增楼的区域面积, 用栈依次弹出, ( 最高的一个乘以1,次高的乘以2...) 乘以的数就是此时i与要计算楼的间隔. 如果栈空,则说明只有最低的那个, 就乘以此时的i就行(前面全都可以截断).

题解

def largestRectangle(h):
    # Write your code here
    stack = []
    area = i = 0
    h.append(0)
    
    # main logic
    while i<len(h):
        # stack is empty or greater height is found
        if not stack or h[stack[-1]] < h[i]:
            # push the index of the building
            stack.append(i)
            i += 1
        # if height is lesser
        else:
            top = stack.pop()
            area = max(area, h[top] * (i - stack[-1] - 1 if stack else i))
    return area
