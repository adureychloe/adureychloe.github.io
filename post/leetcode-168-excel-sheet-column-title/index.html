Leetcode 168: Excel Sheet Column Title

Posted on 2023-09-23，1 min read

题目描述

给定一个数字，返回excel中相应的列名。如

A -> 1
B -> 2
...
Z -> 26
AA -> 27
AB -> 28

思路

可以看出是根据字母表来取数，应该除以26得到结果和余数，对结果不断迭代。divmod函数能得到结果和余数的元组。

代码

def convertToTitle(self, columnNumber:int) -> str:
    result = []
    while columnNumber:
        columnNumber, res = divmod(columnNumber-1, 26)
        result.append(chr(ord('A')+res))
    return ''.join(reversed(result)) # 列表是从外到内，需要反转一下
