借助Unicode处理中文字符

Posted on 2022-06-27，2 min read

1、判断是否是汉字

Unicode中编码在"\u4e00"与"\u9fff"之间的字符都属于汉字。

def is_chinese(ch):
    if '\u4e00' <= ch <= '\u9fff':
        return True
    return False

2、判断是否是分隔符

import unicodedata
def is_punctuation(ch):
    if ch in ("\n", "\t", "\r") or unicodedata.category(ch) in ("Zs", "Zp", "Zl"):
        return True
    return False

3、判断是否是标点符号

分成两部分：跟据ASCII判断是否为英文标点符号；根据Unicode判断是否为中文标点符号。

import unicodedata
def is_punctuation(char):
    cp = ord(char)
    if(cp >= 33 and cp <=47) or (cp >= 58 and cp <=64) or (cp >= 91 and cp <=96 ) or (cp >= 123 and cp <= 126):
        return True
    cat = unicodedata.category(char)
    if cat.startswith("P"): # 根据Unicode的通用类型
        return True
    return False

4、全角和半角的转换

Unicode中半角字符的编码范围是：33(0x21) ~ 126(0x7E)，全角字符的编码范围是：65281(0xFF01)~65374(0xFF5E)。全角字符和相应的半角字符相差65248(0xFEE0)，利用这一点可以实现二者的互换。

def strQ2B(ustring):
    rstring = ""
    for uchar in ustring:
        inside_code = ord(uchar)
        if inside_code == 12288: # 如果是全角空格，则直接转换
            inside_code = 32
        elif (inside_code >= 65281 and inside_code <= 65374):
            inside_code -= 65248
        rstring += chr(inside_code)
    return rstring
