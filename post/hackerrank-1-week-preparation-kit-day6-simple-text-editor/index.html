HackerRank 1 week preparation kit day6: Simple Text Editor

Posted on 2023-03-21，2 min read

题目描述

实现一个简单的文本编辑器。刚开始包含一个空字符串SSS，然后执行Q个行动，行动分4种：

  • 1：append - Append string to the end of .
  • 2：delete - Delete the last characters of .
  • 3：print - Print the character of .
  • 4：undo - Undo the last (not previously undone) operation of type or reverting to the state it was in prior to that operation.

输入：

例子：

思路

用if判断即可，主要是有一个history列表来存储以前的状态，方便4操作回退。还要注意字符串列表 sss 添加元素要用extend，这样子才会添加每一个字符。
注意这里有个坑，如果history采用temp赋值的方式，temp相对于指向了s，后面s如果变化temp也会跟着变。所以我们要用copy()。

a=[['1','abc'],['1','cd']]
s=[]
for op in a:
    if(op[0]=='1'):
        temp = s
        print(temp)
        s.extend(op[1])
        print(temp)
print(s)

output:
[]
['a', 'b', 'c']
['a', 'b', 'c']

题解

from copy import copy
q = int(input().strip())
ops = list()
s = list()
history = []
for q_itr in range(q):
    ops.append(input().split(' '))

for op in ops:
    #print(op)
    if(op[0]=='1'):

        history.append(s.copy())
        s.extend(op[1])
    elif(op[0]=='2'):
        k = int(op[1])
        history.append(s.copy())
        del s[-k:]
    elif(op[0]=='3'):
        k = int(op[1])
        print(s[k-1])
    elif(op[0]=='4'):
        s = history.pop()
