TaskScheduler

Posted on 2023-08-23，3 min read

问题描述

写一个python的类TaskScheduler，根据任务的优先级和分析师进入程序的时间和偏好分配任务给分析师。每个任务会有一个 Task ID、Task Type和 Task Priority。每个分析师都会有 Analyst Name 和 Analyst Preference（TaskType的列表）。

一个分析师可以被分配多个任务，一个任务如果不在任何分析师偏好列表里，就不分配。为了简化，每个分析人员可以为其偏好列表中的每个任务类型最多分配1个任务。任务分配结果按给出的任务类型列表的顺序输出。您在类中应该有插入任务、插入分析人员和分配函数。请随意使用__init__函数来添加到您的数据模型中。并添加一个清晰的功能来清理所有的任务。

例如：

#enter the tasks 
insertTask(“TaskType1”, “TaskID1”,“MEDIUM”) to TaskScheduler
insertTask(“TaskType1”, “TaskID2” , “URGENT”) to TaskScheduler
insertTask(“TaskType1”, “TaskID3” ,“MEDIUM”) to TaskScheduler
insertTask(“TaskType2”, “TaskID4” , “HIGH”) to TaskScheduler
insertTask(“TaskType3”, “TaskID5” , “HIGH”) to TaskScheduler
#enter the analysts
insertAnalyst(“Analyst1”, [“TaskType1”, “TaskType2”]) to TaskScheduler
insertAnalyst(“Analyst2”, [“TaskType1”,]) to TaskScheduler
insertAnalyst(“Analyst3”, [“TaskType2”]) to TaskScheduler
#call function assign
TaskID2 -> Analyst1
TaskID1 -> Analyst2
TaskID3 -> None
TaskID4 -> Aanlyst1
TaskID5 -> Analyst3

思路

感觉题目实例结果应该不对，应该是首先按优先级排序，再按分析师的先后顺序和偏好决定。类里面应该有一个任务列表和一个分析师字典，三个函数：插入任务、插入分析师、分配任务。分析师字典存储名字和偏好列表。还应该有一个字典存储分析师已分配任务。

代码

from collections import defaultdict

class TaskScheduler:
    def __init__(self):
        self.tasks = []
        self.analysts = {}

    def insertTask(self, taskType, taskId, priority):
        pri2id = {'LOW':1,'MEDIUM':2, 'HIGH':3, 'URGENT':4}
        self.tasks.append((taskType, taskId, pri2id[priority]))

    def insertAnalyst(self, name, preferences):
        self.analysts[name] = preferences

    def assign(self):
        # Sort tasks by priority
        self.tasks.sort(key=lambda x: x[2], reverse=True)

        # Track assigned tasks
        assigned = defaultdict(list)

        # Iterate through tasks
        for taskType, taskId, priority in self.tasks:
            # Find analyst with this task type in preferences
            candidates = [a for a,p in self.analysts.items() if taskType in p and len(assigned[a]) < len(p)]
        
            if candidates:
                # Assign to first candidate
                analyst = candidates[0]
                assigned[analyst].append(taskType)
                print(f"{taskId} -> {analyst}")

            else:
                # No candidates
                print(f"{taskId} -> None")

scheduler = TaskScheduler()

scheduler.insertTask("TaskType1", "TaskID1", "MEDIUM")
scheduler.insertTask("TaskType1", "TaskID2", "URGENT") 
scheduler.insertTask("TaskType1", "TaskID3", "MEDIUM")
scheduler.insertTask("TaskType2", "TaskID4", "HIGH")
scheduler.insertTask("TaskType3", "TaskID5", "HIGH")

scheduler.insertAnalyst("Analyst1", ["TaskType1", "TaskType2"])
scheduler.insertAnalyst("Analyst2", ["TaskType1"])  
scheduler.insertAnalyst("Analyst3", ["TaskType2"])

scheduler.assign()
