HackerRank 10 Days of Statistics: Mean, Median, and Mode

Posted on 2023-08-18，1 min read

题目

第一行给定一个数字N代表列表元素个数，第二行给出 N 个空格分隔的整数。求这些整数的平均数、中位数和众数。注意返回值是浮点数的话保留1位小数，众数有多个返回最小的那个。

代码

n = int(input())

def mean(num):
    return sum(num)/n

    
def median(num):
    if(n%2==0):
        med = (num[n//2]+num[n//2-1])/2
    else:
        med = num[n//2]
    return med

def mode(num):
    idx = {}
    for i in num:
        if i not in idx:
            idx[i] = 1
        else:
            idx[i]+=1
    max_time = max(idx.values())
    modes = [i for i,v in idx.items() if v == max_time]
    mode = min(modes)
    
    return mode

num = sorted([int(i) for i in input().split()])
me = mean(num)
med = median(num)
mod = mode(num)
print(f"{me:.1f}\n{med:.1f}\n{mod}")
