Leetcode 21. Merge Two Sorted Lists

Posted on 2023-07-12，1 min read

问题描述

两个有序链表，将它们合并成一个有序链表。

思路


算法逻辑类似拉拉链，用一个新的指针连接两个链表，用到“虚拟头结点”技巧，也就是dummy节点，相当于一个占位符，可以避免处理空指针的情况，降低复杂性。

题解

class Solution {
public:
    ListNode* mergeTwoLists(ListNode* l1, ListNode* l2) {
        ListNode* dummy = new ListNode(-1), *p = dummy;
        ListNode* p1 = l1, *p2 = l2;
        while(p1!=nullptr && p2!= nullptr){
            if(p1->val>p2->val){
                p->next = p2;
                p2 = p2->next;
            }
            else{
                p->next = p1;
                p1 = p1->next;
            }
            p = p->next;
        }
        if (p1!=nullptr){
            p->next = p1;
        }
        if (p2!=nullptr){
            p->next = p2;
        }
        return dummy->next;
    }
};
