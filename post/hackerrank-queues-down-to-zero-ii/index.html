HackerRank Queues: Down to Zero II

Posted on 2023-04-01，2 min read

题目描述

给定Q个query，每个query包含一个单独的数字N，对N可以执行以下2个动作：

  • N能被分解成a×ba \times ba×b（a和b不是1），改成N=max(a,b)N = max(a,b)N=max(a,b)
  • N减1

返回需要将N减至0的最小步数。

输入第一行是数量Q，下面是query。

例子

思路

这里不对2个操作进行比较，而是用队列存储当前数字和已走过步数，对两个操作都走一遍，看谁先结束。注意先进行第二个操作，在进行第一个操作的for循环。因为第一个操作更快速接近结果，要用for循环找遍所有可能的情况，再进行第二个操作。

只有n改变后count才会继续加，代表接下来走的路。（count一样n不一样的节点代表不同的路径）

题解

from math import sqrt
from collections import deque

def downToZero(n):
    # Write your code here
    memo = set()
    count = 0
    q = deque([[n, count]])
    
    # main logic
    while q:
        n,count = q.popleft()
        # base case
        if n <= 1:
            if n==1:
                count+=1
            break;
        
        # 2nd operation
        if n-1 not in memo:
            memo.add(n-1)
            q.append([n-1,count+1])
        
        # 1st operation
        for i in range(int(sqrt(n)),1,-1): # max factor is sqrt(n)
            if n%i == 0:
                factor = max(n//i, i) # max(a,b)
                if factor not in memo:
                    memo.add(factor)
                    q.append([factor, count+1])
    return count
