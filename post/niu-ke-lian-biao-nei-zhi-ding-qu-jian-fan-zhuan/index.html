牛客 链表内指定区间反转

Posted on 2023-04-26，2 min read

题目描述

将一个链表从m位置到n位置之间的区间反转。

例如：1->2->3->4->5，m=2, n=4。
反转后 1->4->3->2->5。

思路

还是要用3个头指向前、现在和后节点。但是cur指向的节点每次往后移动一位，prev指向节点一直是m前一个（不变的那个），他们指向节点保持不变。next指向cur后一个节点，随着cur位置变化而改变指向节点。所以一直是cur往后移动，next指向节点变成prev下一个，从而完成反转。

题解

ListNode* reverseBetween(ListNode* head, int m, int n) {
        // write code here
        // if(head==nullptr){
        //     return nullptr;
        // }
        ListNode* res = new ListNode(-1);
        res->next = head;
        ListNode* prev = res;

        ListNode* cur = head;
        //cur = cur->next;
        
        //ListNode* next = nullptr;

        for(int i=1;i<m;i++){
            prev = cur;
            cur = cur->next;
        }

        for(int i=m;i<n;i++){
            ListNode* next = cur->next;
            cur->next = next->next;
            next->next = prev->next;
            prev->next = next;
        }
        return res->next;
    }
